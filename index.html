Используемые технологии

  1. HTML5
    HTML

    - это язык разметки для создания веб-страниц, который определяет и организует содержание и структуру страницы.

  2. CSS3
    CSS

    - это язык, который определяет, как должны выглядеть элементы на веб-странице, включая их цвет, размер, расположение и т.д.

  3. JavaScript
    JavaScript

    - это язык программирования для создания интерактивных и динамических элементов на веб-сайтах: всплывающие окна, калькуляторы, слайдеры и т.д.

  4. Gulp
    Gulp

    - это средство автоматизации задач, упрощающее процесс сборки, оптимизации и обработки файлов проекта.

  5. Git
    Git

    - это инструмент, который помогает сохранять и отслеживать изменения в коде и совместно работать над проектами.
